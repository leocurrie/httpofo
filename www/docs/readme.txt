Atari Portfolio Web Server
==========================

This web server runs on an Atari Portfolio palmtop
computer from 1989, connected via SLIP over a serial
port at 9600 baud.

Technical specs:
- Intel 80C88 CPU @ 4.9 MHz
- 128 KB RAM
- 40x8 character LCD display
- MS-DOS 2.11 compatible

The network stack implements:
- SLIP (RFC 1055)
- IP (RFC 791)
- ICMP Echo (RFC 792)
- TCP (RFC 793)
- HTTP/1.0

Pretty cool for 1989!
